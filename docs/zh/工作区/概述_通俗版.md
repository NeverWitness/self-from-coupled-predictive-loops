# 概述

这个模型的出发点很简单：我们平时所说的“我”，也许并不是一个固定存在的东西，而更像是一条在时间中不断延续的轨迹。

大脑一直在做三件事情：一方面，它会产生各种可能的行为和状态变化；另一方面，它会不断预测接下来会发生什么；同时，它还会把过去的经历压缩成一个可以持续使用的整体结构。这些过程并不是分开的，而是在同一个系统里同时运行。

在这样的过程中，系统其实隐含着很多可能的发展路径。但在每一刻，只有一条路径能够被稳定地延续下去。那条路径的特点是，它和过去的状态最一致，也最符合系统自己的预测。可以把它理解为“最顺的一条路”。这条不断被延续的路径，就构成了我们体验到的“我”。

体验本身也可以用类似的方式理解。大脑内部同时在进行预测、比较和整合，但这些过程都指向同一个状态，又在同一时间发生，而且没有一个独立的“观察者”去区分它们的来源。结果就是，这些不同的过程在主观上被融合成一个整体，于是我们感觉到的是一个统一的体验，而不是很多分散的信号。

“我在想”或者“我在做”的感觉，也可以从这里产生。当系统的预测和实际发生的事情保持一致，并且这种一致性在时间上持续下去时，这条轨迹就被不断确认，于是系统自然会把这些行为归到同一个来源上，也就是“我”。如果这种一致性被打破，这种归属感也可能变得不稳定。

从这个角度看，自我不是一个隐藏在系统里的实体，而是一个不断被选出来、被维持住的过程。它之所以显得稳定，是因为系统总是在寻找那条最连续、最一致的路径；它之所以显得统一，是因为内部的各种信号在结构上无法被区分。

用一句话来概括，这个模型的想法是：我们所体验到的“我”，其实是一条被持续维持的时间轨迹，而我们的体验，则是这条轨迹上多个过程融合在一起后的结果。
