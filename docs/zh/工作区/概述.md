# 概述

本框架提出，自我与主观体验可以理解为一个预测驱动系统中的结构性产物：系统在持续生成行为候选（可包含扰动）、预测自身状态演化（预测）、并对历史状态进行压缩表示（记忆）的过程中，形成一组时间上的状态轨迹。系统在任一时刻并非处于单一轨迹，而是隐含多个潜在轨迹，但通过持续的预测—演化一致性约束，会选择连续性误差最小的一条轨迹作为主导轨迹。该轨迹在时间上的稳定延续即构成“自我”，可形式化为对轨迹空间中连续性误差的最小化过程。

在此结构中，所谓主观体验并不对应独立过程，而是多个内部信号（预测、匹配、整合）在同一状态变量上的耦合结果。这些信号在时间上高度同步、在表示空间中共指同一对象、且缺乏独立通道或外部参照，因此在系统内部不可分辨，从而在功能上表现为单一体验。换言之，体验的统一性来源于信号的不可分解性，而非其单一性。

主动性（agency）在该框架中并非单一预测成功的结果，而是一个**多预测竞争与归因选择过程**：系统会对自身行为生成多个可能的预测轨迹，实际状态演化与其中一个预测保持一致。当这一一致性在压缩轨迹中维持连续时，该行为被归因于当前主轨迹，从而表现为“由我产生”。相反，当预测失败或连续性误差显著增大时，归因机制可能偏离当前主轨迹，使行为或思维被解释为外源。

这些“多个预测”的来源可以具有不同机制，包括但不限于：预测过程本身的取样特性、对潜在扰动的建模与估计、扰动对系统状态或预测过程的影响，以及上述因素的组合。因此，多预测结构是系统内在生成的结果，而非必须依赖外部扰动。

该模型的核心约束在于：系统趋向最小化跨时间的轨迹偏差，从而维持单一稳定轨迹；多轨迹并存会引入预测冲突并降低整体稳定性，因此被抑制。这一机制同时解释了自我的唯一性、连续性以及在预测与可能扰动之间维持的动态稳定结构。需要强调的是，自我与主观体验的形成并不依赖扰动的存在，而是源于预测—匹配—压缩结构本身。

由此，自我可以被视为一个在时间中被持续选中的轨迹，而非独立实体；主观体验则是该轨迹上多源信号不可分辨融合的结果。

形式上，可将该框架压缩为三条对应关系：
自我对应于连续性误差最小的轨迹，
体验对应于不可分辨的信号耦合，
主动性对应于多预测中的匹配选择与归因机制。
