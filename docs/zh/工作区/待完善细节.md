# 待完善细节（Open Issues & To Be Refined）

## 一、时间尺度分层问题

当前模型隐含所有过程在同一时间尺度运行，但实际上存在多层时间结构：

* 快速环（毫秒级）：匹配 / 预测误差
* 中速环（10–100毫秒）：状态预测
* 慢速环（秒级及以上）：轨迹压缩与更新

需要明确：

> 多时间尺度过程如何叠加，并在主观上表现为“同时发生”

## 二、连续性误差（Continuity Error）的定义

当前使用“continuity error”作为核心变量，但尚未形式化。

待明确：

```id="w3o3vn"
E = distance(当前状态, 上一时刻预测状态)
```

```id="rbr7a3"

```

需要进一步定义：

* 时间窗口范围
* 是否加权
